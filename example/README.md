# example

This directory contains example code demonstrating how to use the `xbrl-go` library for parsing and extracting data from XBRL documents.

```
├── main.go # Example main program
├── sample.xsd # Sample XBRL schema file
└── sample.xbrl # Sample XBRL file for testing
```

## Usage

To run the example, navigate to this directory and execute:

```shgo
go run main.go
```
